## Overview
This project is the second case study from the 
[Google Data Analytics Professional Certificate](https://www.coursera.org/professional-certificates/google-data-analytics) program. The analysis focuses on Bellabeat, a wellness technology company that develops smart products designed to help women monitor and improve their health.

The primary objective of this project is to analyze smart device usage data and identify behavioral trends that can help Bellabeat better understand its customers. These insights are then used to support data-driven marketing recommendations and business growth opportunities.

Using Python as the primary analytical tool, this project follows the complete data analysis process including data cleaning, transformation, exploratory analysis, visualization, and recommendation development.

The analysis explores user behavior patterns such as:
- Daily activity levels
- Step counts
- Sleep habits
- Calories burned
- Weekly activity trends

The final result of this project includes insights, visualizations, and strategic recommendations designed to help Bellabeat improve customer engagement and enhance its marketing strategy.

---

## Business Task

Bellabeat wants to better understand how consumers use smart fitness devices.

The objective of this project is to identify trends in smart device usage and provide marketing recommendations that can help Bellabeat attract, engage, and retain more customers.

## Data Source

Dataset: Fitbit Fitness Tracker Data

Source: Kaggle (Mobius)

The dataset contains fitness tracker data from 30 Fitbit users, including:

- Daily activity
- Steps
- Calories burned
- Sleep tracking
- Activity intensity

---

## Data Cleaning & Processing

The following steps were completed during the cleaning process:

- Removed duplicate records
- Checked for missing values
- Standardized date formats
- Merged activity and sleep datasets
- Created calculated fields for analysis
- Verified data consistency and accuracy

---

## Key Questions

1. What trends exist in smart device usage?
2. How do users engage with fitness and wellness tracking features?
3. What relationships exist between activity, sleep, and calorie expenditure?
4. How can these insights support Bellabeat's marketing strategy?

---

## Key Insights

- Users with higher activity levels generally burn more calories.
- Activity patterns vary throughout the week.
- Many users do not consistently reach recommended activity levels.
- Sleep and activity behaviors show meaningful relationships.

---

## Recommendations

### 1. Personalized Activity Challenges

Encourage users to reach daily activity goals through customized challenges and achievement tracking.

### 2. Promote Wellness Features

Highlight sleep monitoring and wellness insights as key product benefits.

### 3. Data-Driven Marketing Campaigns

Use behavioral data to deliver personalized recommendations and targeted marketing content.
